FIT9136 Introduction to Python

课程简介

  • 本课程会系统介绍 Python 编程语言的核心概念与应用，包括数据类型、控制结构、函数、模块以及面向对象编程。
  • 课程通过项目和练习帮助学生提升编程能力，并训练用代码解决实际问题的思维方式。

主要内容

  • Python 基础语法
  • 递归
  • 面向对象编程、正则表达式与 unittest

评估总结

  • Assignment 1（solo project）：25%
  • Assignment 2（group project）：30%
  • Assignment 3（solo project）：15%
  • TEST 1 in class：10%
  • TEST 2 in class：10%
  • TEST 3 in class：10%
  • 整体考核由三次作业和三次课堂测试组成。第一个作业没有 interview，通常更容易拿到较高分数；后两个作业会加入 interview，难度会明显上升。即使是小组作业，也建议按 solo 作业的标准准备，因为 coding 和 interview 都需要自己真正理解。课堂测试主要考察现场手写代码能力。

学习建议

  • 先把基础概念学扎实，明确每一段代码在做什么、为什么这样写。
  • 多做编程练习，尤其是手写代码训练，这对提升编码能力和逻辑思维非常重要。

常见难点

  • 面向对象编程、正则表达式和小组作业通常是最容易卡住的部分。
  • 小组作业的难度会大幅提升，尤其是 interview 部分。建议提前准备，确保自己能清楚解释代码逻辑。
  • 考试主要考察现场写代码的能力，建议在接近考试的阶段多做限时练习，尽量模拟真实考试环境。
