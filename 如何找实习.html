如何找实习

建立求职基础

  • 找实习不是只靠投简历，而是一个长期准备过程。你需要同时积累编程能力、项目经验、算法训练和对行业的基本理解。
  • 建议从 LeetCode 基础题开始刷起，先建立常见数据结构和算法题型的手感，再逐步提高难度。
  • GitHub 项目很重要。项目不一定一开始就很复杂，但要能清楚展示你解决了什么问题、用了什么技术，以及你在其中承担了什么工作。

作品与简历

  • 简历要突出技术栈、项目成果和可量化的贡献，避免只写“参与”“学习”“了解”这类模糊表达。
  • 项目 README 要认真写，至少包括项目简介、技术栈、运行方式、功能截图或演示链接。
  • 如果目前没有实习经历，可以用课程项目、个人项目和开源贡献补充，但要保证内容真实、可解释。

学习路径

  • 先打好 Python、数据库和基础数学，再补充一门更贴近工业开发的语言或框架，例如 Java、C++、JavaScript、React 或后端框架。
  • 持续刷题，但不要只刷题。项目、简历、面试表达和对岗位的理解同样重要。
  • 推荐观看 with Jiayi Weng talking。其中关于计算机行业和个人评价体系的讨论，对建立求职世界观很有帮助。

心态建议

  • 计算机行业很强调能力和结果，因此你需要为自己建立一套可执行的评价体系：今天写了什么代码，解决了什么问题，项目推进了多少，简历是否更清楚。
  • 不要因为短期没有回复就否定自己。求职本身有运气成分，但长期看，持续改进作品、能力和表达会显著提高机会。
