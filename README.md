# Monash AI 课程指南（C6007）

这是一个面向 Monash University 人工智能与计算机相关方向学生的中文课程指南网站。项目以静态 HTML 页面形式整理课程经验、学习建议、实习准备和 PhD 申请思考，适合跨专业入门、规划学习路线、准备作业与考试的同学参考。

## 项目内容

- `index.html`：网站首页，包含前言、课程目录和延伸主题入口。
- `fit9136.html`：FIT9136 Introduction to Python 课程指南。
- `fit9132.html`：FIT9132 Introduction to Database 课程指南。
- `MAT9004.html`：MAT9004 Math for AI and DS 课程指南。
- `fit5125.html`：FIT5125 IT Research 课程指南。
- `如何申请PhD.html`：PhD 申请准备与风险评估。
- `如何找实习.html`：实习准备、项目、简历与学习路径建议。
- `image.png`：项目中使用的图片资源。

## 如何查看

本项目是纯静态网页，不需要安装依赖或启动后端服务。

在线访问地址：<https://jackcheng030910.github.io/Monash-AI-C6007/>

直接用浏览器打开 `index.html` 即可查看首页：

```bash
open index.html
```

也可以将整个目录部署到 GitHub Pages、Netlify、Vercel 或其他静态网站托管平台。

## 维护方式

新增课程页面时，可以参考现有 HTML 文件的结构：

1. 创建新的 `.html` 页面。
2. 保持统一的页面样式和标题结构。
3. 在 `index.html` 的课程目录中添加入口链接。
4. 如有必要，在本 README 的项目内容部分补充说明。

建议每个课程页面包含以下信息：

- 课程简介
- 主要内容
- 评估总结
- 学习建议
- 常见难点
- 推荐资源
